# URL-Shortener

A URL-Shortener created using Node-JS and synced with Mongo Database.

## Why use it?

Great tool for shortening a long URL.

## Features

* Shorten a Long URL using the web app.
* Gets saved in Mongo database for quick access.

## Built With

* node js (express js)
* mongo_db
* javascript
* html
* css
* bootstrap v5
* ejs
* mongoose
